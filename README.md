# dcl-mathematics

*Foundations of the A=1 Discrete Causal Lattice: the Diamond
Progression in Arbitrary Dimension.*

Paper-only supporting-mathematics repository in the A=1 Discrete
Causal Lattice (DCL) series.  Houses the foundational mathematics
of the *diamond progression* -- the family of bipartite octahedral
lattices in dimensions $d \geq 2$ generated by the d-of-(d+1) regular
simplex vertex construction -- and the dimensional-selection
corollary that follows from coupling the diamond progression to a
minimum probability quantum $\delta p_\min$.

In the user's working notes (`physics-research/notes/raw_thoughts.md`)
this is known as **Paper IV**.

## What the paper proves

Three result-density theorems characterising the diamond progression
as a function of $d$:

1. $\text{coord}(d) = 2d$ (linear in dimension)
2. maximum Euclidean hop = $\sqrt{d}$
3. unit-cell volume = $(d+1)^{(d-1)/2}$

And the dimensional-selection corollary:

$$d_{\max} = \frac{1}{\delta p_{\min}} - 1$$

At $\delta p_\min = 1/4$, $d_{\max} = 3$ -- the candidate mechanism
that *forces* the universe to be three dimensional rather than just
permitting it.  The empirical investigation of $\delta p_\min$'s
value lives in the companion repository
[`dcl-delta-p-min`](https://github.com/JackDMenendez/dcl-delta-p-min).

`dcl-mathematics` does not consume `dcl-core` or any package; it is
paper-only.  Boltzmann-shaped operators and the fluid (NS / Euler)
limit appear as *motivating examples* (Section~10), not proof
targets of this paper.

**Part I: Combinatorial Foundations (the Diamond Progression).**

1. Introduction.
2. The diamond progression: d-of-(d+1) simplex vertex construction.
3. Three result-density theorems (coord = 2d, max hop = √d,
   unit-cell volume = (d+1)^((d-1)/2)).

**Part II: Analytic Foundations.**

5. Discrete differential geometry on the diamond progression.
6. Function spaces on the diamond progression.

9. Dimensional selection: $d_{\max} = 1/\delta p_{\min} - 1$.

Plus appendices: Symbols, Terms, Code and Data, Reproducibility

## Build

```sh
./build.sh paper        # POSIX / MSYS2 UCRT64 on Windows
build.cmd paper         # Windows cmd / PowerShell
```

Build requirements:

- GNU Make $\geq$ 4.3 (on Windows: MSYS2 UCRT64 + `pacman -S make`).
- `pdflatex` + `bibtex` (TeX Live or MiKTeX).
